My tools
=====

### tpl.js
    简单模板引擎(Javascript实现)，无任何依赖(可用于浏览器和NodeJS)，高效短小，压缩后仅800+字节

### shell2http.js
    shell命令转http服务，在服务器上起一个http服务，用GET的方式就可以使用命令行。（使用方式：nohup node shell2http.js &）

### bit.c
    按bit操作Buffer（支持快速查找Buffer里首位为1的位置）

### comm.sh
    日常生活中用到的各种Comm Shell命令（简短的功能性命令）

### misc
    一些杂七杂八的东西
